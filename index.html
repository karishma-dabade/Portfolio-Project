Projects

Online Book Store

The capstone project entitled “Online Book Store” website is an online platform designed and developed in HTML, CSS, Bootstrap, JSP, Servlet, Hibernate and MySQL. It was aimed as an alternative in the current process of easy shopping of e- commerce sites is to sell e-Book.

E-commerce Website

A full-featured e-commerce website that allows users to browse, search, and purchase products online. Features include user authentication, product filtering, a shopping cart, secure payment integration, and an admin dashboard for inventory management. Developed using React.js for the frontend, Node.js for the backend, and MongoDB for the database.

Predicting House Prices

A machine learning project aimed at predicting house prices based on various factors like location, size, number of bedrooms, and age of the house. Utilizes regression algorithms like Linear Regression and Random Forest. The dataset is preprocessed to handle missing values, outliers, and categorical variables. Visualizations are created with Matplotlib and Seaborn to explore trends.
